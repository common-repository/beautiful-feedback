== Description ==

Sliding beautiful configurable ajax based feedback form.
Powered by jQuery.

Выезжающая красивая настраиваемая форма обратной связи.
Работает на аяксе, использует jQuery.

== Installation ==

1. Upload `beautiful-feedback` directory to the `/wp-content/plugins/` directory
1. Activate the the plugin through the «Plugins» menu in WordPress
1. Additional submenu item will appear in Settings menu
1. Configure feedback form as you like
1. Open your blog and you will see feedback form on the right side
1. You can customize form background and open/close buttons - check images buttin_bg.png and form_clean.png in folder of the plugin

---

2. Загрузите директорию `beautiful-feedback` в директорию `/wp-content/plugins/` на вашем сайте
2. Активируйте плагин в меню «Плагины»
2. В форме меню Настроек появится новый подпункт
2. Задайте параметры формы ны ваше усмотрение
2. Откройте блог, и вы увидите форму справа
2. Вы можете изменить фоновую картинку и надписи открыть/закрыть. Для этого найдите изображения button_bg.png и form_clean.png в папке images плагина
